# 🎲 Ludo Dice Distribution Analyzer

## 📘 Project Description
The **Ludo Dice Distribution Analyzer** is a C programming project that simulates rolling two six-sided dice millions of times (for example, 1,000,000 rolls) to study how often each possible total (2–12) appears.  
The program calculates and displays the **probability distribution** of the dice sums, helping verify whether real random rolls match theoretical mathematical probabilities.

This project demonstrates how randomness and probability work together in a real simulation, providing insight into stochastic modeling and data analysis through programming.

## ⚙️ How It Works
1. The program uses the **`rand()`** function to simulate rolling two dice.
2. Each die returns a random number between **1 and 6**.
3. The program repeats this process **1,000,000 times** using a **for loop**.
4. An **array** is used to count how many times each possible sum (2–12) appears.
5. Finally, the program prints a **formatted table** showing the data of count and probability of each sum.
